# Bloco 02: Git & GitHub

Esse bloco pertence ao módulo de fundamentos do curso da Trybe. Nesse bloco aprendi sobre o Git e como configurar-lo na minha maquina, e também como iniciar um repositório local. Depois aprendi sobre o GitHub e como subir meu repositório local para o GitHub, o que é Branch, Merge e commit. Assim fui capaz de instalar e configurar o Git no meu computador, salvar meus projetos no repositório Git local, controlar as alterações e versões dos meus arquivos e enviar meus projetos para o repositório remoto do GitHub;

Tendo isso em mente aqui você encontrará os exercícios que desenvolvi para fixar os conhecimentos que adquiri ao longo desse bloco.
